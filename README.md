Requisitos

-Node.js (versão recomendada: 18+)

-NPM instalado

## 🛠 Tecnologias Utilizadas

- Vue 3  
- Pinia  
- Vite  
- Vue Draggable (SortableJS)  
- JavaScript (ES6+)  
- Bootstrap 5  

Como executar o projeto localmente

Siga os passos abaixo para rodar o projeto em sua máquina:

1️⃣ Clone o repositório

-[git clone https://github.com/seu-usuario/kanban-main.git](https://github.com/Viniciussvn/kanban.git)
 
 (ou baixe o zip e siga o próximo passo)

2️⃣ Abra o projeto no VS Code

-Abra a pasta kanban-main no Visual Studio Code.

-Após, utilize no terminal (' CTRL + " '):

-cd kanban-main

-Ou manualmente pelo VS Code:
-File → Open Folder → selecione a pasta kanban-main → kanban-main

3️⃣ Instale as dependências

-No terminal do VS Code, execute:

-npm i
4️⃣ Execute o projeto

-Após instalar as dependências, rode:

-npm run dev

5️⃣ Acesse no navegador

-O terminal exibirá uma URL local SEMELHANTE a:

-http://localhost:5173/

-Abra esse endereço no navegador para visualizar o projeto.
